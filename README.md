# Secure Banking Transaction System

## Overview
This project simulates secure communication between a client and a bank server using RSA authentication and Diffie-Hellman key exchange.

## Technologies Used
- Python
- Google Colab
- RSA
- Diffie-Hellman
- Cryptography

## Features
- Client-server transaction simulation
- Secure key exchange
- RSA authentication
- Encrypted and decrypted file exchange

## How to Run
1. Open the notebook in Google Colab.
2. Run all cells from top to bottom.
3. Check the generated encrypted and decrypted files.
